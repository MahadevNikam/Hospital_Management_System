# Hospital Management System

A Hospital Management System built using Java, JDBC, and MySQL to streamline basic hospital operations such as managing patients, doctors, and appointments.

## Features

* Add, view, update, and delete patient records.
* Add and manage doctor information.
* Book and manage appointments between patients and doctors.
* User-friendly menu-driven console interface.
* Data stored and managed using MySQL database.
* JDBC integration for database connectivity and operations.

## Tech Stack

* Programming Language: Java
* Database Connectivity: JDBC
* Database: MySQL
* IDE: VS Code / IntelliJ IDEA / Eclipse

## Key Concepts Used

* Object-Oriented Programming (OOP)
* Classes and Objects
* Encapsulation
* Exception Handling
* JDBC Connection Management
* SQL Queries (INSERT, SELECT, UPDATE, DELETE)

## Project Objective

The objective of this project is to automate basic hospital management tasks, reduce manual record keeping, and provide an efficient way to manage hospital data through a simple Java application.
